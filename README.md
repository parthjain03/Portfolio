# Parth Jain — Portfolio (Next.js)

This repository is a minimal, premium-feeling portfolio site built with Next.js (pages router). It includes:
- Light / dark mode (persisted)
- Clean typography and airy layout
- Per-project pages with download links and photo placeholders
- Static hosting friendly — ready for Vercel

## Quick local run
1. Install dependencies: `npm install`
2. Run dev server: `npm run dev`
3. Open `http://localhost:3000`
